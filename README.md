# Personal Agile Planner

Учебный проект для рейтинговой работы по дисциплине "Проектная деятельность в ИТ".

Вариант по первой букве фамилии: "Разработка программы планировщика Agile для личного пользования".

## Назначение

Personal Agile Planner - простая консольная программа для личного планирования задач по Agile-подходу. Программа предназначена для ведения небольшого списка задач, распределения их по статусам и контроля приоритетов.

Цель проекта - показать полный цикл создания небольшого ИТ-продукта: от описания назначения и требований до реализации, тестирования и фиксации этапов разработки в Git.

## Стек

- Python 3
- Стандартная библиотека Python
- JSON-файл для хранения данных
- CLI-интерфейс

## Планируемые возможности

- создание задачи;
- просмотр списка задач;
- хранение задач в JSON;
- распределение задач по статусам: backlog, todo, in_progress, done;
- указание приоритета и срока выполнения;
- базовая проверка логики с помощью тестов.

## Пользователи

Основной пользователь программы - студент или индивидуальный пользователь, который планирует личные учебные, рабочие или бытовые задачи. В учебной версии предусмотрен один уровень доступа: владелец локального списка задач.

## Функциональные требования

- программа должна запускаться из командной строки;
- программа должна позволять добавить задачу с названием, описанием, приоритетом и сроком;
- программа должна сохранять задачи в файл `data/tasks.json`;
- программа должна выводить список сохраненных задач;
- каждая задача должна иметь статус Agile-доски: `backlog`, `todo`, `in_progress` или `done`;
- проект должен содержать базовый тест, подтверждающий создание и сохранение задачи.

## Запуск

```bash
python -m src.agile_planner.main --help
```

Добавление задачи:

```bash
python -m src.agile_planner.main add "Prepare report" --priority high --due-date 2026-05-10
```

Просмотр задач:

```bash
python -m src.agile_planner.main list
```

Запуск тестов:

```bash
python -m unittest discover -s tests
```
